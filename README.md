# Примитивная реляционная СУБД

Проект реализует простую реляционную систему управления базами данных на Python с поддержкой SQL-подобного синтаксиса, валидацией типов данных и хранением в JSON-файлах.

## Установка

```bash
# Установка зависимостей
make install

# Запуск СУБД

```bash
make project
# или
poetry run project
